# 🧪 QA Testing Project - Travfix Web

## 📌 Deskripsi

Project ini merupakan portfolio Quality Assurance (QA) yang mencakup proses testing secara end-to-end pada aplikasi web Travfix.

Testing dilakukan untuk memastikan kualitas sistem melalui:

* Manual Testing
* API Testing
* Automation Testing

---

## 🎯 Scope Testing

Fitur yang diuji dalam project ini:

* Login
* Booking
* Dashboard
* Ticket Search

---

## 🛠 Tools & Technology

* Python (Selenium)
* Postman
* Jira
* Microsoft Excel
